# RESTful API
A RESTful API for Car Rental use with validation/authentication (<b>Node.js</b>, <b>Express.js</b>, <b>MongoDB Atlas</b>) <b>- MVC</b>

Featuring <b>CRUD</b> operations for authenticated users!

-----------------------------------------------------------------------------------------------------------
<b>Instructions</b>:

1. Rename .env.example file to .env and use your credentials
2. Whitelist your IP ( https://cloud.mongodb.com Network Access -> IP Whitelist -> + ADD IP ADDRESS )
-----------------------------------------------------------------------------------------------------------
